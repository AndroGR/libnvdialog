Download NvDialog and
start using it today!

NvDialog is available primarily through the GitHub releases.
The current latest release is 0.2.0, while
you can also download and compile directly through Git.

Need prebuilt binaries?

v0.3.0 added prebuilt libraries in the Releases (Currently unreleased).
You should be able to download them with the new release.

Prebuilt binaries are only available for Windows and Linux. For other platforms,
you will have to compile NvDialog yourself.

Compiling from source:

The documentation shows how to manually compile the library
from source code. This may be useful mainly for previous releases
if the new version of the library has an issue.

You are adviced strongly, however, to open an issue on GitHub
instead, so the bug can be resolved instead and be able to still use
nvdialog with its latest version and features.
