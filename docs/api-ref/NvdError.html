NvDialog API Reference

API version: 0.1.3



Declaration 


/* Declaration of NvdError. */
typedef enum {
        NVD_NO_ERROR = 0,
        NVD_NO_DISPLAY = 0xff,
        NVD_INVALID_PARAM,
        NVD_NOT_INITIALIZED,
        NVD_STRING_EMPTY,
        NVD_OUT_OF_MEMORY,
        NVD_INTERNAL_ERROR,
} NvdError;

Brief Description

Return type of nvd_get_error. Represents an error code set by NvDialog.

Detailed Description

An enumerator indicating the error NvDialog currently has encountered. This code is only returned by nvd_get_error, and can be stringified using the nvd_stringify_error function.
The enum starts at 0xff and NVD_NO_ERROR (0) is not considered an error.

Example Code 


#include <nvdialog/nvdialog.h>
int main(int argc, char *argv[])
{
        NvdError result = nvd_init(argv[0]);
        if (!result) {
                /* No error, continue. */
        } else { /* Initialization failed */
                /* If not initialized, then print an error and exit. */
                printf("Failed to initialize NvDialog: %s\n", nvd_stringify_error(result));
                exit(EXIT_FAILURE);
        }
        return 0;
}
