NvDialog API Reference

API version: 0.1.3



Declaration

/* Function Declaration */
NvdVersion nvd_get_version(void);

Brief Description

Returns the version of NvDialog the user is using.

Detailed Description

This function returns the current linked version of NvDialog at runtime. You will have to manually extract each field from the function's return type (NvdVersion).

Parameters

This function takes no parameters.

Example Code 


#include <nvdialog/nvdialog.h>
int main(int argc, char *argv[])
{
        NvdVersion ver = nvd_get_version();
        printf(
                "NvDialog linked version: %d.%d.%d",
                ver.major,
                ver.minor,
                ver.patch
        );
        return 0;
}
