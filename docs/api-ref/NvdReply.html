NvDialog API Reference

API version: 0.1.3



Declaration 


/* Declaration of NvdReply enumerator. */
typedef enum {
        NVD_REPLY_OK = 0x04,
        NVD_REPLY_CANCEL,
        NVD_REPLY_NO
} NvdReply;

Brief Description

An enumerator holding the return value of an nvd_question_box_new.
On C++, you may also use raw integers if it suits your needs.

Detailed Description

This is an enumerator that is used as the return type of nvd_question_box_new. See the function documentation for more information or return values.
As the names imply, the enum returned represents the button pressed.

Example Code 


#include <\stdlib.h\> // Backslashes not needed !
int main(int argc, char *argv[])
{
        if (nvd_init(argv[0]) != 0)
                abort();
        else {
                NvdReply reply = nvd_question_box_new(
                        "Some title",
                        "Some question you want to ask",
                        NVD_YES_NO_CANCEL
                );
                if (reply == -1) {
                        /* Something went wrong, handle error */
                } else {
                        if (reply == NVD_REPLY_OK)
                                /* User pressed OK */
                        else if (reply == NVD_REPLY_CANCEL)
                                /* User cancelled */
                        else if (reply == NVD_REPLY_NO)
                                /* User pressed no */
                        else
                                /* Something bad happened */
                }
        }
        return 0;
}
