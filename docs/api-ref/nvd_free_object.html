NvDialog API Reference

API version: 0.3.0



Declaration

/* Function Declaration */
void nvd_free_object(void* object);

Brief Description

Deletes an object creates by NvDialog.

Detailed Description

Call this function when you are no longer interested in using the parameter passed anymore, to free up any resources occupied by the object. Note that calling this in the middle of an operation will cause undefined behavior.

Parameters

object -- The object to be deleted.

Example Code 


#include <nvdialog/nvdialog.h>
#include <stdlib.h>
#include <stdio.h>

int main(int argc, char *argv[]) {
        nvd_init(argv[0]);
        NvdDialogBox* dialog = nvd_dialog_box_new("Example", "Example Box", NVD_DIALOG_SIMPLE);
        if (!dialog) abort();

        nvd_show_dialog(dialog);
        nvd_free_object(dialog); // No longer used, so we can just free it.
        return 0;
}
