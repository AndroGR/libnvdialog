NvDialog API Reference

API version: 0.1.3



Declaration

/* Function Declaration */
int nvd_init(char* program);

Brief Description

Initializes NvDialog's backends and sets internal flags to appropriate values.

Detailed Description

This function initializes the internals of NvDialog as well as
its backends (Gtk3 / Gtk4). Not calling this function before any other call
to NvDialog will result in undefined behavior, ranging from overflows to distorted UIs.

Parameters

program
This should be the argv[0] parameter of your executable.
Passing this parameter incorrectly will result in undefined behavior.

Example Code 


int main(int argc, char *argv[])
{
        if (nvd_init(argv[0]) != 0) {
                exit(EXIT_FAILURE);
        }
}
