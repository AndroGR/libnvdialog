NvDialog API Reference

API version: 0.3.0



Declaration

/* Function Declaration */
void nvd_delete_parent(void);

Brief Description

Unmarks the window set from nvd_set_parent as the parent window.

Detailed Description

This will reset the changes made by nvd_set_parent, by unmarking the window that is given as the parent from NvDialog.

Parameters

No parameters.

Example Code 


#include <nvdialog/nvdialog.h>
#include <stdlib.h>
#include <stdio.h>

int main(int argc, char *argv[]) {
        nvd_init(argv[0]);
        GtkWindow* window = gtk_window_new(GTK_WINDOW_TOPLEVEL);

        nvd_set_parent((void*) window);

        GtkWindow* window = (GtkWindow*) nvd_get_parent();
        if (!window) abort();
        nvd_delete_parent(window);

        return 0;
}
