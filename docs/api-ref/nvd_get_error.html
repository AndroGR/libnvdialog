NvDialog API Reference

API version: 0.3.0



Declaration

/* Function Declaration */
int nvd_get_error();

Brief Description

Returns the current error code of the library.

Detailed Description

The error code has to be manually be transformed into a string using nvd_stringify_error. Else, use the NvdError enum to check for the error manually.

Parameters

No parameters.

Example Code 


#include <nvdialog/nvdialog.h>
#include <stdlib.h>
#include <stdio.h>

int main(int argc, char *argv[]) {
        nvd_init(argv[0]);

        if (nvd_get_error() != 0) return 1;
        else return 0;
}
