NvDialog API Reference

API version: 0.3.0



Declaration

/* Function Declaration */
const char* nvd_get_error(NvdError err);

Brief Description

Transforms an error code into a string representation, that can be used to print errors to the console. Recommended: Enable automatic logging of the library's errors instead of manually logging them yourself.

Detailed Description

(Empty)

Parameters

err -- The error code enum value.

Example Code 


#include <nvdialog/nvdialog.h>
#include <stdlib.h>
#include <stdio.h>

int main(int argc, char *argv[]) {
        nvd_init(argv[0]);
        
        if (nvd_get_error() != NVD_NO_ERROR) {
                const char* err = nvd_get_error();
                printf("Error: %s\n", err);
                exit(EXIT_FAILURE);
        }

        return 0;
}
