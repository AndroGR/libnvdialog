NvDialog API Reference

API version: 0.3.0



Declaration

/* Function Declaration */
const char* nvd_get_argv(void);

Brief Description

Returns the argv[0] parameter set during initialization of NvDialog.

Detailed Description

This function will return the argv[0] parameter set during initialization, or NULL if no initialization happened. If the latter is the case, feel free to ignore it.

Parameters

No parameters.

Example Code 


#include <nvdialog/nvdialog.h>
#include <stdlib.h>
#include <stdio.h>

int main(int argc, char *argv[]) {
        int result = nvd_init(argv[0]);
        if (result != 0) {
                puts("Couldn't initialize NvDialog, aborting execution.");
                abort();
        }

        const char* argv_0 = nvd_get_argv();
        printf("argv_0: %s\n", argv_0);

        return 0;
}
