Welcome to NvDialog's documentation!

NvDialog offers documentation about building and using the library
in order to ensure developers get familiar with it. The documentation does not cover the API
reference; If this is what you are looking for, click here.

Also, for future contributors, make sure you get familiar with the directory layout of
nvdialog. This way consistency is kept nice and tidy, and you also save everyone alot of time.
