1. Getting Started with NvDialog:

Hello and welcome to NvDialog! The very first step is how to install NvDialog
so you can start using it at your projects. Installing NvDialog is a relatively simple
process, however on fresh systems you will need to install a bit of software first.

More specifically, you will need to install:

  • CMake (To build NvDialog)
  • Ninja (To execute CMake's commands)
  • Any modern C compiler (Preferably GCC or CLang)
  • Git (You will only need this if you want the Git version)

Assuming you installed all of these, and you have extracted the source code somewhere (Or cloned it), enter the source directory (eg. cd nvdialog/), and generate the build files with the following commands: 


$ cmake . -B build/
$ cd build/

After doing so, in order to compile NvDialog, you should use Ninja, with the generated files present in the directory: 


$ ninja

This step is enough if you aren't planning to install NvDialog, but just use it for
a single project. From there on, you can put the compiled library where you wish, instruct
the compiler to find the NvDialog headers, and you should be good to go.

Otherwise, you should also run this command as the superuser on your system: 


$ ninja install

These are all the steps for building and installing NvDialog. In order to test if
NvDialog is properly working, you can build the examples under the examples/
directory and run them. Alternatively, you can simply run the example from the homepage.
